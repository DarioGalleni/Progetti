gulpfile.js - File di configurazione Gulp
.git/ - Cartella di configurazione Git
node_modules - Dipendenze npm

Solo i file essenziali necessari per far funzionare il tuo sito web:

File HTML (come index.html)
File CSS compilati (nella cartella css)
File JavaScript (nella cartella js)
Immagini (nella cartella imgs e sottocartelle)
Font e icone (nella cartella themify-icons)
File SEO importanti (robots.txt e sitemap.xml)
